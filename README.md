# Krit Anti-Gravity Student Project Package

Status: student-project handoff, research draft  
Prepared: 2026-06-18

This package turns the anti-gravity material into a buildable student project for Krit at Chiang Mai University. The generated handoff files are the sendable material. Raw source copies can be kept locally for provenance or sent later on request.

The handoff keeps three claim levels separate:

- Conventional acoustic effects: buildable and expected. Piezo plates, pot lids, cymbals, and ultrasonic arrays can produce measurable acoustic radiation pressure, squeeze-film forces, vibration coupling, heat, and electrostatic artifacts.
- OPH chi_nu effect: a hypothesis under test. The proposed signal is a support-force residual tied to a self-reading, sign-reversible top/bottom coherence contrast. This package claims no receipt.
- Human-carrying anti-gravity: excluded from the student scope. The cymbal rig here is a bench force platform and coherence-body test article, not a rider vehicle.

Best first student project:

Build Version 1, the piezo-crystal plate pendulum. It is cheap, safe at low voltage, quick to instrument, and directly tests the measurement discipline needed by the larger cymbal rig.

Best second project:

Build a single-cymbal calibration station before the four-cymbal platform. One cymbal or pot lid can map force versus frequency, phase, drive amplitude, and standoff distance without the mechanical complexity of a four-corner frame.
